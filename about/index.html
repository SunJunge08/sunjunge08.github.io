这两天，我陆续地把之前的大部分文章都做了一次大更新：即把文件和图片存放至主站，停用 GitHub Release 和图床。我为什么要这样做呢？原因有两个。

一是图片存放在图床上不安全且容易丢失数据。例如，我最开始用的图床是 SM.MS ，但是后来我发现它会无缘无故地删除一些图片。然后，这个图片就会被替换为一张写有“该图片已被删除”字样的图片，还会让我充会员。

二是之前想要下载文件都要通过 GitHub Release ，而国内经常墙掉 GitHub ，所以可能会导致文件无法下载。即使我用了一些 GitHub Release 的加速服务（就是用国外的服务器来进行“代理”下载），也会有加速服务挂掉的风险。到时候将会很麻烦。

有了以上两个原因，我就开始寻找一种方法，能够在不借助第三方平台（指图床平台和 GitHub 等，虽然这个 Blog 也是在 GitHub 的基础上搭建的，但是貌似和 GitHub 的服务器无关）的情况下，直接提供图片和文件的传输和下载。换句话说，就是要让这个 Blog 本身自带图床，自带像 GitHub Release 的文件下载功能。于是，我在网上查找资料，终于发现了一种可行的方法。它是这样实现的——

首先，在本地的 source 文件夹下新建一个文件夹（当然也可以建很多个），为其命名，例如存放图片的叫做 images ，存放文件的叫做 download ，至于叫什么都无所谓，自己怎么方便怎么来。

1. 上网课

2022年3月19日晚上 21:15 时，我收到了这样一条消息（有稍微改动）：

各位家长，大家好：

突如其来的一场疫情打乱了我们的学习计划，现在孩子们的身体状况基本好转，孩子们的内心世界慢慢平复，学校准备初二年级在下周二（3月22日）开始网课。知道隔离中的孩子们仍然面临很多困难，请各位家长帮助孩子们做好准备。

  1. 有的隔离地点没有 WiFi，请各位家长给孩子手机充值，我们即使用流量也要保证孩子上课，因为时间最宝贵；
  2. 电源不充足的地方，叮嘱孩子提前充电，或者找到同学共享一个手机视频，或者两个手机交替使用，保证上课时电源充足；
  3. 没有书本的同学，找医护人员借用一些笔记纸，关注老师讲课内容，提前下载教材内容，或者把教材内容拍照，保证上课时手中有参考资料；
  4. 因为咱们的老师仍然有人生病住院，所以咱们联播上课，家长和孩子在一起的，尽量陪同。

希望家长引导孩子们调整好身心，尽快适应即将开始的线上学习，提高网课的学习质量，我们期待早日校园相见。

上网课是必然的，我也早有准备，从家带了台笔记本电脑，可惜忘了带鼠标。上面这条消息中提到的问题，其实我都能克服，而且这都不是什么大问题。真正困扰我的是，大红土没有桌子，这就导致我的笔记本电脑没有地方放置，我没有地方写字。这必然会对网课的质量造成影响。于是，在2022年3月20日中午，我、张爷爷和 S 三个人决定利用屋子里的物件创造出一个大桌子。我们先后实行了三个方案：

方案一：利用“两点确定一条直线”这一公理，在两张床之间的放上一块正方形铁板，如图所示（灵魂画手）。这样我们就能在铁板上写字了。但是，这个铁板面积太小，而且不太稳定，经常掉下来。所以方案一被否定了。

方案二：把阳台的鞋柜放倒，在上面铺上被子和铁板。显而易见，铁板会来回乱动，而且鞋柜放倒后高度太低，放在哪都不合适。

方案三：把屋里的铁柜放倒，有把手的一面朝上，把这一面当做桌子。

这第三套方案比前两套好多了：一是因为铁柜的长度和我的身高差不多，铁柜的宽度和床的宽度差不多，这么大的面积足够我们在上面放置各种物品；二是因为铁柜表面光滑平整，不像被子+铁板那样不稳定；三是我们可以把东西放入铁柜的格子里，节省空间，同时也能支撑上方的压力。但是，它的美中不足就是缺少配套的椅子。说句实在话，在大红土这种地方，是不可能得到桌子和椅子的，只能就地取材。于是，我们把目光投向了先前发给我们的塑料箱子。这个箱子下面带有滚轮，方便我们来回乱动。由于它是塑料制成的，肯定不能直接坐上去，否则就会像我一样把箱子坐裂开。不过这点困难难不倒我们，我们在箱子中填入被子，然后坐在被子上，根据 p=F/S 可知这样会使压强减小，不会使箱子碎裂。

上网课，一个绕不开的问题就是网络。前面的日志中提到过，大红土自带的 WiFi 极其拉胯，因此我只能自己开流量包硬刚。我先是开了一个 10GB30元 的流量包。后来我又在联通的客户端中看到了一个名为“阿里定向流量 15G 包”的流量包，它的描述是这样的（节选）：

2. 新成员

2022年3月20日晚上九点左右，我们206寝室迎来了一名新成员，就叫他 J 吧。他是一个五十多岁的叔叔，带着一副眼镜。刚进屋时，他穿着一身防护服，和我在15日的样子差不多。我让他自己选择床位，他毫不犹豫地选择了我所在位置的上铺。就这样，他正式成为了206寝室的一员。

J 刚躺在床上玩手机，就问我们有没有 WiFi。我告诉他有，但是都被拉黑了，没被拉黑的在屋里也连不上。得知这件事的他并没有像我和 S 一样懊恼失落，而是说他有的是流量，根本不慌。这时张爷爷走了过来，问他一个月多少流量。J 说他对这方面不太懂，不知道有多少流量，但是绝对够用。一听这话，张爷爷十分高兴，说自己手机的流量快要用完了，正愁没有流量用呢。如果 J 愿意开热点，我们就不用担心流量不够用的问题了。J 爽快地答应了开热点的请求，不一会，一个叫做 HONOR Play4T Pro 的热点就出现在了可用 WLAN 列表里，密码是 206206206，一看就是我们寝室的，非常有特色。这个热点果然很好用，连上它，我可以流畅地在 B 站观看 4K60FPS 的视频（我实则是在炫耀自己的大会员），还可以在网课上迅速地回答老师的问题，然后得到老师的点名表扬，再也不怕高 ping 战士了。事实上，J 的流量直到2022年3月24日早上才被用光。我们几个人五天不停地大量消耗流量，才把 J 的流量用光，可以说是非常持久了。据我估计，J 的流量至少有 100GB！

2022年3月21日晚上，我们寝室又加入了一名成员，就叫他 K 吧。K 是吉林市某中学的初三年级学生，他绝对是个重量级人物（物理上），所以张爷爷和 J 都叫他“小胖子”。K 和 J 不同的一点是，K 和我们都是2022年3月15日来到大红土的，后来他的室友要么回家了，要么换屋了，最后只剩下了他一个人，于是就把他安排到了我们这屋。有意思的一点是，他由于太过重量级，只能睡下铺。于是我就慷慨解囊地把自己的位置让给了他，自己搬到了 S 的上铺。此时206已经有五个人了，🎵又回到最初的起点🎵。

还有一个人是我不能不提到的，她是我的表姐，但她并不是206的新成员，而是大红土的新成员。2022年3月20日晚上六点左右，她坐车抵达了大红土，开始了为期14天的监狱生活。在我的不断引（song）导（yong）下，她巧妙地来到了6号楼。但是由于医护人员和管理人员太拉胯，她直到九点才住进来，是在327室。后来她又换了几个屋，最终定居在317室。这几天，她经常在“大红土隔离群”与我和 S 聊天，并交流病情。由于我们不在同一楼层，平时也不怎么见面，所以对她的介绍就先到这里。

3. 离开大红土

在2022年3月18日和21日，我分别进行了两次核酸检测。在2022年3月22日和21日（我没写反），这两次核酸分别出了结果，都是阴性。这就意味着，我马上可以离开大红土回家了！

2022年3月23日晚上，我们2楼的管理人员在群里统计了一下这两次核酸检测结果都是阴性的人数。2022年3月24日上午11点左右，我接到通知，可以走了。于是我赶快收拾东西，飞一般地冲出了206寝室。在一楼，我填写了一张“回家证明”，带着它，我走出了6号楼的大门。临走前，医护人员用乙醇给我洗了把脸，让我十分上头。长达十天的煎熬，我终于来到了6号楼下的一个小广场，等待着客车把我们送回家。我终于重获了自由！这时，我的脑海里出现了一句歌词，“是的我看见到处是阳光 / 快乐在城市上空飘扬 / 新世纪来得像梦一样 / 让我暖洋洋”，出自朴树的 NEW BOY，我那时兴奋而充满期待的心情无以言表。

在楼下，我站了很久，于是我拍了一张照片，如图所示。这就是我带回来的所有物品（不包括书包）。

我又看向206寝室所在的窗户，不一会，S 他们也从窗户中看到了我，如图所示。

远处，一辆绿色公交车慢慢驶来，我终于要离开这里了。临走前，我为6号楼拍了一张照片。

最后，再看一眼这所学校吧。

中午12点左右，我就到了越山路附近，在那里我所在的社区派来了一辆客车把我们所有来自我们社区的人都接了回去。然后，我平安地回到了家。

4. 阴转阳？

2022年3月25日早上，我突然得知我在23日做的一次核酸是阳性，这可把我吓坏了。但没关系，家里有自测盒，我用自测盒检测了一下，结果是阴性。当天社区就来了两批人为我做核酸，其中一批的结果已经出来了，是阴性。但是另一批的结果还不知道。如果它的结果是阳性，那么我大概率会重返大红土。这是我极不情愿的。

5. 总结

到这里，我对自己的隔离生活就已经叙述得差不多了，大概用了一万一千多字，算是一篇比较长的随笔了吧。回想一下，从2022年3月4日开始，疫情就一直围绕在我的身边，直到现在可能还未离去。现在，206寝室里剩下的四个人（S、张爷爷、J 和 K）都尚未具备回家的条件，希望疫情早日结束，我们都能回归正常生活。让我们期待着！

2022年3月19日 星期六

1. 来到大红土

2022年3月15日早晨八点左右，我被叫醒去集中隔离。我赶紧收拾好东西，把作业打印出来，到小区准备走。到了那里，我一把东西放下，就有一位“大白”给我扔过来一套防护服。我从来没穿过防护服，旁边也没有人，他们不会靠近我，所以我只能自己摸索着穿上。

一套防护服有以下几个结构：①连体的上衣+裤子；②一双鞋套；③一副橡胶手套；④一只 N95 口罩；⑤一个护目镜；⑥一个塑料挡板。我在穿防护服的过程中，由于用力太大，把衣服上的拉链弄断了。还好后来有人提醒我用拉链旁边的双面胶替代拉链，这样我才穿好衣服。

与此同时，我刚穿完衣服，就被叫到了大客车里。我和同一小区的两个高中生（这里就说他们是 A 和 B 吧）一起上的车，由于我带的东西太多（我带了一个书包，一个装书的手提包和两个大包，加在一起大概比我还重），他们就帮我把我的两个大包带上了车。

大约过了两三个小时，我们来到了一个非常偏僻的职业院校区，后来我才知道这里叫“大红土”。由于来到这里的大客车实在是太多了，我们又等了好长时间才下车。

2. 来到这里的第一天

2022年3月15日中午，我们整车的人都下了车，走到“吉林机电工程学校”的6号宿舍楼。在进门前，我们每个人都领了一个很薄的“被子”和一个板砖形状的枕头。有一说一，监狱的装备都比这好。

这时发生了一个小意外：我本来是要和 A、B 两人一起走的，可是领被子枕头的时候是五个人一组，而我正好和他们不在一组，于是选择寝室时，我和他们分离了。最开始，我随便找了个没人的寝室，房间号是225，然后满楼去找 A、B 两人，与此同时他们也在找我。幸运的是，我们很快就找到了对方，然后我把我的物品从225搬到了他们的房间415，并在那里铺好被子。

过了亿会，我们吃了午饭，这里的饭菜还不错，我把四个菜都吃光了，真香！吃完饭，我了解到 A 和 B 都是十八中的学生，而且 B 还要上网课。接着，A 联系到了他的一位同学 C，并且没过多久 C 就住到了225寝室。又过了亿会，A 又叫来了他的另一个同学 D。这时，225已经有五个人了。

晚上，我和家长通了一次电话。他们听到了 A 的咳嗽声，觉得很不安全，容易传染，就想让我搬走。我本来是不想走的，但是因为我和他们没有什么共同语言，并且 D 也一直在咳嗽，就有了换寝室的想法。但是我往哪里搬呢？碰巧的是，我得知我的同学 S 也在6号楼，于是我就给他打电话，联系到了他，还搬到了他的寝室206，后来，我一直住在206，没有再搬走过。

当天晚上我还得知，我还有一个同学 E 和一个老师 W 也在大红土，但可惜他们并不在6号楼，我们无法换楼，也不能走到楼外，所以不能见到他们。

3. 隔离风波

我们206寝室里还有另外几个人，我就把他们记为 F、G、Z 吧。

他们的关系是这样的：F 是一位阿姨，G 是她的父亲，Z 是 F 和 G 的邻居，和 F 一起来照顾 G，因为 G 有半身不遂和脑出血，需要有人来照顾。

2022年3月16日上午，F 因为大红土环境太差没有坐便，导致 G 不能正常上厕所，非常着急。她让社区把坐便器带来了，可是大红土的卡点并不允许外人进入，所以他们一直都收不到坐便器。

中午，F 终于忍不住了，她找来医护人员，要求派来一辆救护车，把他们送到医院。终于，在她的据理力争下，她如愿以偿地和 G 转到了中心医院，而 Z 因为救护车的位置不够，就与我和 S 一起留在了大红土。

再来简单介绍一下 Z 吧。他是一位六十多岁的姓张的爷爷，从这里开始，我就称他为“张爷爷”而不叫他“Z”了。经过四天的相处，我认为张爷爷是一个前卫的人——不仅言行前卫，思想也很前卫。对于他的具体情况，我放到后面说吧。

4. “大红土起义”

这算是我来到大红土几天以来最激动人心的一件事了。2022年3月16日中午，由于大红土的环境实在是太差了，吃饭扔掉的垃圾直接堆积在厕所旁，看起来又脏又乱。更重要的是，我们的生活根本没人管（除了每天的一日三餐有人给送），楼道根本不消毒。最最重要的是，我们来的时候没有做核酸检测，所有人不管情况如何都直接被送到这里隔离，这实在是太不负责任了！一位住在2楼某寝室的叔叔实在是忍不住了，他趁着 F 和医护人员争论的时候，把这里的情况用手机拍摄了下来，还找了一位医护人员，问了他很多问题，并拍了视频。之后，他剪辑出了一个视频，准备“曝光”这里。实际上，他的目的只是为了得到更多人关注，从而改善这里的环境。

事实证明，这是有效的。这几天，大红土的环境明显得到了改善。医护人员建立了一个微信群，用来解决我们的各种问题（但是，这个微信群其实没什么作用，我们只能得到口罩和一些简单的洗漱用品，而且还限量，所以实际上这个群根本没什么用）。好在终于有人对我们重视了起来，但还是没人给做核酸，我相信我们的隔离生活还会得到进一步的改善！

由于这件是整个过程都是在走廊里进行的，双方对线十分激烈。我们最终取得了“胜利”。于是，S 把这件事称为“大红土起义”。

5. 关于张爷爷

这里再来介绍一下目前和我们一个寝室的张爷爷吧。我在前面说过，张爷爷是一个很前卫的人。这里我再说几个事例来证明这句话吧。

我刚来到206寝室时，也就是2022年3月15日晚上，那是我正在忙着收拾东西并铺床。这时 F 想要给手机充电，但是电源在很高的位置，需要爬到上铺才能插上充电器，十分地不方便。我见状把自己从家里带的插座给她，这样就能在下面充电了。这时，张爷爷凑上前去，但他的关注点不是这个电源，而是电源旁的另一个插口。那个口大概是插网线的吧，我在家里见过，但是没有使用过。他说他本来想从家里带一根网线来着，可惜他忘记了。大红土虽然有电力供应（听说有的楼还没有，我们比较幸运），但是没有覆盖 WiFi 网络，所以我们只能使用流量。张爷爷说，如果这个网线插口能够使用，就可以不需要密码而连接到网络，结果他还没带网线，太可惜了！我有些惊讶，按理说这个年纪的老人应该不懂这些“先进”的网络知识的，可是他却讲得头头是道，让我有点纳闷，还以为他是看起来显老。

第二天晚上，也就是2022年3月16日，寝室里只剩下我们三个人，我们闲着没什么事干。这时张爷爷就向我们展示了一下他的手机，我一看，上面有颜色不同的几条折线，下面是密密麻麻的数字，看起来是股票之类的东西。他说这不是股票，但是和股票类似，具体叫什么我也不记得了。后来，他还说他在上世纪90年代就开始上网了，算是老客户，知道很多东西，甚至办理宽带的人都没有他知道得多。这才解释了我心中的疑惑。

以上讲的都是他的过去，再来说说他的前卫思想是如何在我们的隔离生活中发挥作用的吧。

2022年3月16日下午，前面讲过，F 和 G 去了医院，剩下我们三人。本来我和 S 都住在上铺，即使他们走了，我们也不想费劲去搬到下铺，虽然下铺真的很方便。这时，张爷爷说，如果我们仍然留在上铺，把下铺空出来，那么一旦以后有人搬到6号楼，一看到我们206寝室空出来两个下铺，就会直接搬进来，而我们三人都不想有人再搬进来。于是，我和 S 就把东西搬到了下铺。

2022年3月17日中午，张爷爷担心外面的人在我们寝室的门口大声喧哗，影响我和 S 学习，就提议让我们做一张告示，贴在门玻璃上。于是我拿了两张 A4 纸，用胶棒粘在一起，然后在上面写上字，我写的是“请勿打扰，影响学习，谢谢”，然后再用胶棒把这两张纸粘到门上，下面留出1/3的空白，以便于我们查看外界情况。

这两件事一起灵验了。2022年3月17日晚上，我和 S 正在水池边洗漱，这时2楼门口来了一群“大白”，他们并不是医护人员，而是将要搬到这里隔离的患者。我们当然不想让他们来到我们206寝室，但是也没法说，就一边洗漱一边观察着这群人的情况。结果上来就是个重量级人物：那个人（我就记为 H 吧）一到二楼就说，他要到206室。那时他已经走到了206的门口。而我和 S 就站在他的旁边。于是我就对他说：“二楼人都住满了，五楼还有好几个房间没有人住，你到五楼去吧。” S也随声附和，在我们两人的不断怂恿下，他走向楼梯，可能是去了五楼吧。结果没过一会他就回来了，和一名医护人员说自己要去206，然后被两个小孩怂恿着去了五楼，现在自己应该怎么办。医务人员当然不傻，他直接开始敲206的门。此时，206屋里只有张爷爷一个人，但他那时好像在睡觉，而且睡得很沉。所以，他压根没听见敲门声。实际上，我们206的门是没有锁的，但是由于门把手的构造问题，外面的人想要成功转动门把手是很困难的，需要使劲往外拽一下然后向左拉一下，很奇怪。而在寝室里面的人却能够轻易地打开门。也是由于这个原因，几乎没有几个医护人员能直接打开我们206的门，我们甚至怀疑那些医护人员是不是手无缚鸡之力。最后，H 还是换了一个寝室住。又过了一会，有好几个大白要住到2楼的各个寝室，可能他们事先就被安排好了吧，后面那一堆人都没有住到我们206寝室，期间，有几个人想要进我们寝室看看，却因为玻璃上的那两张纸而看不到里面的情况，就离开了。当我们睡觉的时候，有几名医护人员又来敲门，可是这次他们不走了，一直在敲。我们没办法只好开门。他们一看，下铺都住满了人，而上铺放了一堆袋子，从门口的角度来看，就像是放满了东西一样，于是他们就离开了。最终，我们206寝室没有新增人员，还是我们三个人。试想，如果没有门口那张告示，外面的人会离开吗？如果我们不搬到下铺，医护人员会离开吗？多亏了张爷爷的神机妙算，才使得我们的隔离生活的质量得到了保障。

最后再说一件事，那就是在2022年3月18日，我们6号楼竞选“楼长”。张爷爷当上了楼长，他帮助楼里的患者们解决了很多问题，还帮忙搬东西，发三餐。更重要的是，他总能带回来一些好东西，例如消毒水、口罩等。此外，他还总能多带回来一两份饭、水果或酸奶给我和 S，针不戳。

说到这里，我就已经把我对张爷爷的印象全部说出来了。感谢张爷爷！

6. WiFi 和微信群

2022年3月17日，楼内的医护人员建了一个微信群，名字叫“2楼急救”。这个群的作用就是解决2楼患者们的各种问题。但实际上，这个群用处不大。我们只从他们那里要了一包口罩，而其余的需求他们都无法满足。比如，我想要一个写作业用的桌子，他们没有；我想要一根上网课用的网线，他们也没有。其他人的需求也是如此。

于是，住在220寝室的一位来自吉林市某高中的老师（记他为 X 吧）建了一个群，名字叫“大红土隔离群”。但是由于这个群并不是医护人员建立的，所以人数不多，只有少数几个寝室的人加了进来。

这里还有一段小插曲——我在“2楼急救”的群里看到了一个人（记他为 Y 吧），其微信头像和我们的历史老师非常相似。我就加了他的微信，一问才知道，Y 是历史老师的丈夫，他带着他的儿子一起来到大红土隔离，还把历史老师的微信推给了我。碰巧的是，他们就住在204，离我们的寝室很近。并且，Y 也在“大红土隔离群”中。

继续说回到 X 和他建立的“大红土隔离群”上吧。由于他的群没有多少人进来，他就想了个损招：在“2楼急救”群中发了一条消息“进这个群，告诉你们这儿的 WiFi 密码”，还在后面加上了“大红土隔离群”的二维码。这消息一发出去，立刻就有很多人进群。与此同时，我和 S 走到了 X 的寝室门口，发现了一个 WiFi，名为 zhijiaoyuan。这时，S 告诉我，他通过手机软件连上了这个 WiFi，但是不知道密码。聪明的我让 S 把这个 WiFi 的二维码发给我，我一扫，出现了如下字符串：

聪明的我一眼就看出来，这个 WiFi 的密码是 yq654321，连接一下，果真如此。并且，我们又发现了另外一个 WiFi，名称为 HUAWEI_STK_bfd2，我随便输入了个 12345678，没想到连上了，这密码也太简单了吧！后来，我们又发现了两个稳定的 WiFi，分别是 HUAWEI P30 和 OPPO Reno 5G，这两个 WiFi 的密码都是 12345678。但是，到了2022年3月18日早上，我发现除了 zhijiaoyuan，其余的 WiFi 要么把我的设备拉黑，要么经常连接不到，所以也没法用。目前，我们只能把希望寄托在 zhijiaoyuan 上了。

7. 总结

这篇日志是2022年3月18日和2022年3月19日这两天一起写完的，总共5000多字，算是我有史以来写过的字数最多的一篇文章了吧。最近开始了几轮核酸检测，目前还没出结果。如果我阳转阴的话，就可能转入医院治疗。有一说一，那里的环境比大红土强几百倍！期待未来会更好！

2022年3月10日 星期日

1. 关于“关于”页面 禁止套娃

这个 blog 是在 2021年11月22日 创建的。目前为止，我已经写了9篇日志。但是这个 blog 一直都没有一个“介绍自己”的页面。如果我单独写一篇文章，显得不够重要。于是，我把它放到了这个 关于 / About 页面中，让这个特殊的页面完成这一任务。

具体来说，这个页面算是我的一个“日记本”吧，单独发表的那些 blog 可以说是我“拿得出手的”文章。一些细碎的、不重要的事情，我会把它们记在这里，方便日后查看。最近我有一些时间，可以写一写，因为最近真的有很多事情！等到我开学后，不能做到经常更新 blog ，到时候我会把几篇日志一起发出去。

2. 我的近况

在 2022年2月中旬 发布的一些 blog 的最后，我说过 “这篇是寒假最后一篇 blog，后续更新只能抽时间慢慢写了” 这样的话语。开学后，一切似乎都很正常，考试、讲课、刷题……好像并没有什么异常。然而，2022年3月3日（星期四）的早上，我们老师突然让全班同学带上了口罩，这样的画面，我入学一年半以来似乎还没有见到过。3月4日 - 9日我的情况如下：

3月4日，学校停课，上了一天线上课，这天我进行了第一次核酸。

3月5日，休息了一整天，没留多少作业，生活过得就像假期一样。

3月6日，早上进行了第二次核酸。晚上说是要把我带走隔离，收拾好东西后等了一晚上也没来。

3月7日，没上课。由于6日晚上睡得很晚，7日上午睡了半天觉（这句话不要被老师看到）。下午写作业，晚上甚至不需要传到小组群，震惊膈肌一整年。

3月8日，过着和7日差不多的生活。晚上有人上门给我做第三次核酸，这次是捅鼻子，感觉很上头。

3月9日，小组群开始了钉钉直播写作业。

3月10日早上7点，我打开电脑在钉钉上自习。中午11点，自习结束，开始午休。这时我第三次的结果出来了，是阳性。这是我早已预料到的，因为在这之前，我们班就已经有好几名同学确诊了，所以其他人很有可能确诊。

晚上9点多，社区发来信息说“可能会把我带走”，于是我赶紧开始收拾东西。然后我去洗了个澡。然而直到晚上11点都没有消息。所以今晚大概率是走不了了。

3. 总结

说了一千多字（水字狂魔），把我要说的基本上都说完了。还有一些事情等到以后慢慢写吧。就这样，希望疫情早点过去，我的书放在学校现在还没取回来呢。
